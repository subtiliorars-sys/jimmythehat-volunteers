# Game Artist (2D/3D)

**Role Description**

---

## Overview

Game Artists create visual assets for our racing game, including sprites, models, textures, UI elements, and more. This is a learning-focused role with mentorship from experienced artists.

## Responsibilities

### Core Tasks
- Create 2D sprites and textures
- Design UI elements and menus
- Create 3D models (if applicable)
- Optimize assets for game performance
- Follow the project's art style and direction

### Learning Opportunities
- Build a portfolio of game art
- Learn game-specific art techniques
- Understand asset optimization
- Practice working within style constraints

## Requirements

### Artistic Skills
- Basic drawing or digital art skills
- Understanding of color theory and composition
- Familiarity with digital art software
- Ability to follow style guides

### Technical Skills
- Knowledge of file formats (PNG, SVG, etc.)
- Understanding of asset optimization
- Familiarity with version control (Git) helpful but not required

### Soft Skills
- Good communication skills
- Ability to take feedback constructively
- Creativity and attention to detail
- Reliable and consistent

### Time Commitment
- 5-10 hours per week
- Minimum 2-month commitment preferred
- Flexible scheduling

## What You'll Work On

### Example Tasks
- Create character sprites
- Design UI buttons and menus
- Create environment textures
- Design power-up icons
- Create vehicle models (3D)
- Animate characters or objects

### First Project
Your first task will be small and well-defined, such as:
- Creating a simple icon
- Modifying an existing asset
- Designing a UI element

## Tools and Technologies

### Recommended Software
- **2D Art:** Photoshop, GIMP, Krita, or similar
- **3D Art:** Blender, Maya, or similar (if applicable)
- **Vector Art:** Inkscape, Illustrator, or similar
- **Version Control:** GitHub for asset sharing

### File Formats
- **Sprites:** PNG (with transparency)
- **UI Elements:** PNG or SVG
- **3D Models:** FBX, OBJ, or GLTF
- **Textures:** PNG or JPG

## FAQ

**Do I need to be a professional artist?**
No! We welcome artists of all skill levels. The most important thing is passion and willingness to learn.

**What software do I need?**
Any digital art software you're comfortable with. We can recommend free options if needed.

**Can I choose what I work on?**
To some extent. We'll assign tasks based on project needs, but we'll consider your interests and strengths.

**How do I share my work?**
You'll upload assets to the project repository via GitHub. We'll help you learn if you're not familiar with it.

**Can this lead to paid work?**
While we can't guarantee paid work, the experience and portfolio pieces can help you get paid positions elsewhere.
