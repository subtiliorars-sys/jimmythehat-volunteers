# Introduction Email — To Accepted Applicants

Sent once an application is accepted. Personalize the `[bracketed]` bits.

---

**Subject:** Welcome to JimmyTheHat Games — your next steps 🥜🏁

Hi [Name],

Welcome aboard! We reviewed your application for the **[Role]** role and we're glad
to have you on Driving Me Nuts.

This is a volunteer / educational project — no pay, but real repo experience, code
review from mentors, and school-credit documentation if you need it.

**Your first steps**
1. **Accept your GitHub invite** — check your email from GitHub (org: `subtiliorars-sys`).
   Repo to clone: https://github.com/subtiliorars-sys/DrivingMeNuts---Preview
2. **Read the getting-started guide:** https://github.com/subtiliorars-sys/jimmythehat-volunteers/blob/main/docs/onboarding/getting-started.md
3. **Join our Discord:** [invite link] — say hi in `#welcome`.
4. **Pick your first task:** https://github.com/subtiliorars-sys/jimmythehat-volunteers/blob/main/docs/GOOD_FIRST_ISSUES.md
   Comment on the linked issue to claim it, and a mentor will assign you.

**Logistics**
- Commitment: ~5–10 hrs/week. Flexible — just communicate if life gets busy.
- Under 18? Return the signed parental consent form (attached) before you start coding.
- Questions anytime: reply to this email, open a [Question issue](https://github.com/subtiliorars-sys/jimmythehat-volunteers/issues/new?template=question.yml), or ask in Discord.

We're excited to build with you. See you in the repo!

— The JimmyTheHat Games team
volunteers@jimmythehat.com
