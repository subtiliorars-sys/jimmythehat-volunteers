# DrivingMeNuts - Project Overview

**Peanut Food-Truck Racing Game**

---

## About the Game

DrivingMeNuts is a racing game featuring peanut food trucks. Players race through various environments, collect power-ups, and compete to be the fastest food truck on the road.

### Genre
Racing / Arcade

### Platform
Web (browser-based)

### Tech Stack
- **Language:** TypeScript
- **Engine:** Custom TypeScript game engine
- **Rendering:** HTML5 Canvas / WebGL

---

## Game Features

### Core Gameplay
- **Racing:** Competitive racing against AI or other players
- **Power-Ups:** Various power-ups to gain advantages
- **Vehicles:** Multiple food truck vehicles with different stats
- **Tracks:** Diverse racing environments
- **Progression:** Unlock new vehicles and tracks

### Planned Features
- Multiplayer racing
- Vehicle customization
- Leaderboards
- Achievements
- Seasonal events

## Design Philosophy

### Fun First
Every mechanic should be fun. If it's not fun, cut it or fix it.

### Accessible Depth
Easy to pick up, but with depth for mastery.

### Clear Feedback
Players should always understand what's happening and why.

### Meaningful Choices
Every decision should matter and feel impactful.

## Technical Architecture

### Game Loop
- Update game state
- Handle input
- Update physics
- Render frame
- Repeat

### Systems
- **Physics:** Custom physics engine
- **Rendering:** Canvas-based rendering
- **Input:** Keyboard and gamepad support
- **Audio:** Web Audio API
- **Networking:** WebSockets for multiplayer
