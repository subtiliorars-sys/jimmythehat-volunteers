## Open Roles

### Junior Game Developer (TypeScript)
**Requirements:**
- Basic programming knowledge (JavaScript/TypeScript preferred)
- Interest in game development
- 5-10 hours per week availability
- Age 16+ (with parental consent if under 18)

**Responsibilities:**
- Implement game features based on specifications
- Bug fixing and testing
- Code documentation
- Participate in code reviews

**What You'll Learn:**
- Real game development workflows
- TypeScript best practices
- Game architecture and patterns
- Collaboration in a team environment

### Game Artist (2D/3D)
**Requirements:**
- Basic art skills (digital or traditional)
- Interest in game art
- 5-10 hours per week availability
- Age 16+ (with parental consent if under 18)

**Responsibilities:**
- Create game assets (sprites, models, textures)
- UI design elements
- Character design
- Asset optimization

### Game Designer
**Requirements:**
- Interest in game mechanics and design
- Basic understanding of game development process
- 5-10 hours per week availability
- Age 16+ (with parental consent if under 18)

**Responsibilities:**
- Design game mechanics
- Create level layouts
- Write game design documents
- Playtesting and feedback

## How to Apply

**Option 1: GitHub Issue**
1. Click "Issues" tab above
2. Click "New Issue"
3. Use the "Volunteer Application" template
4. Fill out the form and submit

**Option 2: Email**
Send your application to volunteers@jimmythehat.com with:
- Your name and age
- Role you're interested in

- Your availability

## What Happens Next

1. **Review:** We'll review your application within 48 hours
2. **Screening:** If under 18, we'll send parental consent forms
3. **Interview:** Quick call to discuss the project and your interests
4. **Onboarding:** Set up GitHub access, join our communication channels
5. **First Task:** Start with a small, well-defined task

## FAQ

**Is this paid?**
No, this is a volunteer/educational opportunity. Great for building your portfolio and gaining experience.

**Do I need experience?**
No! We welcome beginners. The main requirement is passion and willingness to learn.

**What if I'm under 18?**
We welcome volunteers aged 16-17 with parental consent. The process includes an additional consent form for your parent/guardian.

**How long is the commitment?**
We ask for a minimum of 5 hours per week for at least 2 months. This ensures you have time to contribute meaningfully.

**Can I get school credit?**
Yes! We can provide documentation of your volunteer hours and contributions for school requirements.

**What if I don't know TypeScript?**
We can help you learn! If you have basic programming knowledge, we'll guide you through TypeScript specifics.
